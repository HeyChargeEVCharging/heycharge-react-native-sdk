# heycharge-sdk

The mobile HeyCharge SDK for React Native platform

## Installation

```sh
npm install heycharge-sdk
```

## Usage

```js
import { multiply } from 'heycharge-sdk';

// ...

const result = await multiply(3, 7);
```
